# MOC-着色

> 图论中**着色理论**的跨章节汇总，涵盖顶点着色、边着色、色多项式及平面图着色。

## 概览

着色问题是图论最经典的研究方向之一。顶点着色与边着色分别研究对点和边的颜色分配，使得相邻对象颜色不同。色多项式则计数着色的方式数。平面图的四色/五色定理是图论最著名的结果之一。

## 顶点着色

### 核心概念

| 概念 | 说明 | 关联wiki |
|------|------|---------|
| [[图的点着色]] | 正常顶点着色、$k$-可着色 | ★★★ |
| [[色数]] | $\chi(G)$，最少颜色数 | ★★★ |
| [[次大度]] | $\Delta_2(G)$，Brooks定理证明参数 | ★★ |

### 重要定理

| 定理 | 内容 | 考试重点 |
|------|------|---------|
| [[Brooks定理]] | 连通非完全非奇圈：$\chi \leq \Delta$ | ★★★ |
| [[五色定理]] | 平面图：$\chi \leq 5$ | ★★★ |

### 色数上界
- 平凡上界：$\chi(G) \leq \Delta(G) + 1$
- Brooks改进：$\chi(G) \leq \Delta(G)$（除完全图和奇圈外）
- 平面图：$\chi(G) \leq 5$（五色定理），实际上 $\chi(G) \leq 4$（四色定理）

## 边着色

### 核心概念

| 概念 | 说明 | 关联wiki |
|------|------|---------|
| [[图的边着色]] | 正常边着色、$k$-边可着色 | ★★★ |
| [[边色数]] | $\chi'(G)$，最少颜色数 | ★★★ |

### 重要定理

| 定理 | 内容 | 考试重点 |
|------|------|---------|
| [[Vizing定理]] | 简单图：$\Delta \leq \chi' \leq \Delta + 1$ | ★★★ 类1/类2 |
| [[偶图的边色数]] | 偶图：$\chi' = \Delta$（König） | ★★★ |
| [[奇阶正则图的边色数]] | 奇阶 $\Delta$-正则图：$\chi' = \Delta + 1$ | ★★★ |

### 边色数分类
- **类1图**：$\chi' = \Delta$（如偶图、平面图 $\Delta \geq 8$）
- **类2图**：$\chi' = \Delta + 1$（如奇阶正则图）

## 色多项式

### 核心概念

| 概念 | 说明 | 关联wiki |
|------|------|---------|
| [[色多项式]] | $P_k(G)$，正常 $k$-着色数目 | ★★★ |
| [[色多项式递推公式]] | $P_k(G) = P_k(G-e) - P_k(G \cdot e)$ | ★★★ |
| [[理想子图]] | 补图中各分支为完全图的生成子图 | ★★ |
| [[伴随多项式与色多项式]] | 利用补图理想子图计算 | ★★ |

## 与特殊图的关联

- **偶图**：$\chi = 2$，$\chi' = \Delta$
- **完全图**：$\chi(K_n) = n$，$\chi'(K_n) = n$（$n$ 奇）或 $n-1$（$n$ 偶）
- **树**：$\chi = 2$，$\chi' = \Delta$
- **平面图**：$\chi \leq 5$，$\chi' = \Delta$（若 $\Delta \geq 8$）
- **正则图**：奇阶正则图是类2图；偶阶正则图可能是类1或类2
- **Euler图**：$\chi'$ 与圈的分解有关

## 与平面图的关联

着色理论与平面图密不可分：
- [[五色定理]] 是平面图着色理论的核心
- 四色猜想（现已证明）是图论最著名的定理
- 平面图的边着色也有特殊结果
