# 色数

> **定义**：对图 $G$ 正常顶点着色需要的最少颜色数，称为图 $G$ 的**点色数**，简称**色数**（chromatic number）。图 $G$ 的色数用 $\chi(G)$ 表示。

![[file-20260423113758998-色数示例证明.png]]

## 性质与特征

由 $\chi(G)$ 及 $P_k(G)$（[[色多项式]]）的定义，有：

1. 若 $k < \chi(G)$，则 $P_k(G) = 0$；$\chi(G) = \min\{k \mid P_k(G) \geq 1\}$
2. 若 $G$ 为 $n$ 阶空图，则 $P_k(G) = k^n$
3. $P_k(K_n) = k(k-1)\cdots(k-n+1)$
4. 若图 $G$ 含有 $n$ 个孤立点，则 $P_k(G) = k^n P_k(G')$

## 基本结论

| 图类 | 色数 | 说明 |
|------|------|------|
| 一般简单图 | $\chi \leq \Delta + 1$ | 平凡上界 |
| 连通非完全非奇圈 | $\chi \leq \Delta$ | [[Brooks定理]] |
| 简单平面图 | $\chi \leq 5$ | [[五色定理]] |
| 简单平面图 | $\chi \leq 4$ | 四色定理（Haken & Appel, 1976）|
| 完全图 $K_n$ | $\chi = n$ | |
| 偶图 | $\chi = 2$ | |
| 奇圈 $C_{2k+1}$ | $\chi = 3$ | |

## 例子

上图展示了一个图及其色数证明：因图中含有奇圈，所以色数至少为 $3$；又因为点 LA 与该圈上每一个点均邻接，所以色数至少为 $4$。另一方面，用 $4$ 种颜色实现了正常点着色，所以图的色数为 $4$。
