# 极图

极图属于==极值图论==的范畴，主要研究满足某个条件下的最大图或最小图问题。

## l部图的概念与特征

![[file-20260422205403950-l部图定义.png]]

> **定义**：若简单图 $G$ 的点集 $V$ 有一个划分：
> $$V = \bigcup_{i=1}^l V_i, \quad V_i \cap V_j = \emptyset, \quad i \neq j$$
> 且所有的 $V_i$ 非空，$V_i$ 内的点均不邻接，称 $G$ 是一个==$l$ 部图==。

### 注
1. 如果 $l=2$，则 $G$ 就是偶图
2. $n$ 阶无环图必是 $n$ 部图
3. 若 $l_1 < l_2 \leq n$，则任意的 $l_1$ 部图也是 $l_2$ 部图

### 完全l部图

> **定义**：如果在一个 $l$ 部图 $G$ 中，$|V_i| = n_i$，任何两点 $u \in V_i$，$v \in V_j$，$i \neq j$，$i, j = 1, 2, \ldots, l$ 均邻接，则称 $G$ 为==完全 $l$ 部图==。记作 $G = K_{n_1, n_2, \ldots, n_l}$。

### 完全l几乎等部图（Turán图）

![[file-20260422205403953-Turán图定义.png]]

> **定义**：如果在一个 $n$ 阶完全 $l$ 部图 $G$ 中，$n = kl + r$（$0 \leq r < l$），$|V_1| = |V_2| = \cdots = |V_r| = k + 1$，$|V_{r+1}| = |V_{r+2}| = \cdots = |V_l| = k$，则称 $G$ 为 $n$ 阶==完全 $l$ 几乎等部图==，记为 $T_{l, n}$。

图 $T_{l, n}$ 的 $n$ 个点的分布为：
$$\underbrace{\left\lceil\frac{n}{l}\right\rceil, \left\lceil\frac{n}{l}\right\rceil, \ldots, \left\lceil\frac{n}{l}\right\rceil}_{r}, \underbrace{\left\lfloor\frac{n}{l}\right\rfloor, \left\lfloor\frac{n}{l}\right\rfloor, \ldots, \left\lfloor\frac{n}{l}\right\rfloor}_{l-r}$$

$|V_1| = |V_2| = \cdots = |V_l|$ 的完全 $l$ 几乎等部图称为==完全 $l$ 等部图==。

### 边数上界定理

**定理**：$n$ 阶完全偶图 $K_{n_1, n_2}$ 的边数 $m = n_1n_2$，且 $m \leq \left\lfloor \frac{n^2}{4} \right\rfloor$。

**证明**：$m = n_1n_2$ 显然。下面证明第二结论：
$$m(K_{n_1, n_2}) = m(K_{n-n_2, n_2}) = (n - n_2)n_2 = \frac{n^2}{4} - \left(\frac{n}{2} - n_2\right)^2 \leq \left\lfloor \frac{n^2}{4} \right\rfloor$$

**定理**：$n$ 阶 $l$ 部图 $G$ 有最多边数的充要条件是 $G \cong T_{l, n}$。

**证明**：首先 $m(G) \leq m(K_{n_1, n_2, \ldots, n_l})$。其次，考虑
$$f(n_1, n_2, \ldots, n_l) = \sum_{i<j} n_in_j, \quad \text{s.t.} \quad \sum_{i=1}^l n_i = n$$
则 $f$ 取最大值的充分必要条件：对任何的 $1 \leq i < j \leq l$，有 $|n_i - n_j| \leq 1$。此时 $G$ 的对应的顶点划分形成的 $l$ 部图正好为 $T_{l, n}$。

> **思考**：图 $T_{l, n}$ 的边数为 $\frac{(l-1)n^2 + r^2}{2l} - \frac{r}{2}$，其中 $n \equiv r \pmod{l}$。
