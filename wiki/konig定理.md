# König定理

> **定理陈述**：在偶图中，最大匹配的中的边数等于最小覆盖中的点数


前置概念：[[匹配]] [[点覆盖]] [[偶图]]

### 证明思路
利用最大匹配 $M^*$ 构造出由 $M^*$ 交错路连接的顶点集合 $Z$，进而得到一个与 $M^*$ 等势的点覆盖，从而证明最大匹配的边数等于最小覆盖的点数。

### 详细证明

**证明**：设 $G$ 是具有二分类 $(X, Y)$ 的偶图，$M^*$ 是 $G$ 的最大匹配。用 $U$ 表示 $X$ 中的 $M^*$ 非饱和顶点的集合，用 $Z$ 表示由 $M^*$ 交错路连接到 $U$ 中顶点的所有顶点的集合。

置 $S = Z \cap X$，$T = Z \cap Y$。

类似于Hall定理的证明，可知 $T$ 中的每个顶点都是 $M^*$ 饱和点，$|S| - |U| = |T|$，并且 $N(S) = T$。

令
$$K = (X \setminus S) \cup T$$

则 $G$ 的每条边必然至少有一个端点在 $K$ 中。否则，存在一条边，其一个端点在 $S$ 中，而另一个端点在 $Y \setminus T$ 中，这与 $N(S) = T$ 相矛盾。因此 $K$ 是 $G$ 的一个覆盖。

又因为
$$|K| = |X| - |S| + |T| = |X| - |U| = |M^*|$$

而由点覆盖与匹配的基本关系知，对任意匹配 $M$ 和任意覆盖 $K$，均有 $|M| \le |K|$。因此
$$|M^*| \le |K_{\min}| \le |K| = |M^*|$$

故 $|M^*| = |K_{\min}|$，即最大匹配中的边数等于最小覆盖中的点数。
