# Vizing定理

> **定理陈述**：设 $G$ 是简单图，则
> $$\Delta(G) \leq \chi'(G) \leq \Delta(G) + 1$$
> 即简单图的边色数要么等于最大度 $\Delta$，要么等于 $\Delta + 1$。

![[file-20260423113729380-Vizing定理陈述.png]]

## 形式化表述

对于任意简单图 $G$，其边色数 $\chi'(G)$ 只有两种可能：
- **第一类图**（Class 1）：$\chi'(G) = \Delta(G)$
- **第二类图**（Class 2）：$\chi'(G) = \Delta(G) + 1$

## 证明

![[file-20260423113729384-Vizing定理证明.png]]

### 证明思路

对边数 $m$ 用数学归纳法。只须证 $\chi' \leq \Delta + 1$。

**基例**：当 $m = 1$ 时，$\Delta = 1$，$\chi' = 1$，有 $\chi' \leq \Delta + 1$。

**归纳步骤**：设 $m \geq 2$，取 $G$ 中一条边 $xy$，令 $G' = G - xy$。由归纳假设知 $\chi'(G') \leq \Delta(G') + 1$。因 $\Delta(G') \leq \Delta(G)$，故 $\chi'(G') \leq \Delta(G) + 1$。于是存在 $G'$ 的正常 $\Delta(G)+1$ 边着色。在 $G'$ 中，对着色 $\pi$，显然每个点至少缺少一种颜色。于是，由引理知 $G' + xy = G$ 是 $\Delta(G)+1$ 边可着色的，从而 $\chi'(G) \leq \Delta(G) + 1$。

## 推论

根据 Vizing 定理，简单图可以按边色数分成两类图：
- 边色数等于 $\Delta$ 的简单图（第一类）
- 边色数等于 $\Delta + 1$ 的图（第二类）
