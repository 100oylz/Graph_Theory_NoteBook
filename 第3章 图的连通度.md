# 第3章 图的连通度

## 本章概览

本章研究图的连通程度及其度量，引入割边、割点、块等基本概念，定义顶点连通度与边连通度，建立它们与最小度之间的关系，并介绍连通性理论的核心结果——Menger定理与Harary图的构造方法。

## 核心概念

### 3.1 割边、割点和块
- [[割边]] — 删去后增加连通分支数的边
- [[割点]] — 删去后增加连通分支数的顶点
- [[块]] — 没有割点的极大连通子图

### 3.2 连通度
- [[顶点割]] — 使图不连通的顶点子集
- [[顶点连通度]] — 最小顶点割的大小，记为 $\kappa(G)$
- [[边割]] — 使图不连通的边子集
- [[边连通度]] — 最小边割的大小，记为 $\lambda(G)$
- [[连通度]] — 顶点连通度与边连通度的统称及基本关系
- [[k-连通图]] — 顶点连通度至少为 $k$ 的图
- [[k-边连通图]] — 边连通度至少为 $k$ 的图

### 3.3 Menger定理与网络应用
- [[Menger定理]] — 分离数与不相交路数目的等价关系
- [[Harary图]] — 最小边数的 $n$ 阶 $k$ 连通图
- [[连通度的应用]] — 通讯网络可靠性模型

## 重要定理
- **割边判定定理** — $e$ 是割边当且仅当 $e$ 不在任何圈中（[[割边]]）
- **割点判定定理** — $v$ 是割点当且仅当 $v$ 位于某条 $(x,y)$ 路的每条路上（[[割点]]）
- **块判定定理** — 阶 $\geq 3$ 的图是块当且仅当无环且任意两点共圈（[[块]]）
- **连通度不等式** — $\kappa(G) \leq \lambda(G) \leq \delta(G)$（[[连通度]]）
- **度条件定理** — $\delta(G) \geq (n+k-2)/2$ 则 $G$ 是 $k$ 连通的（[[顶点连通度]]）
- **Menger定理** — 分离数 = 不相交路的最大数目（[[Menger定理]]）
- **Whitney定理** — $k$ 连通 $\Leftrightarrow$ 任意两点间 $k$ 条独立路（[[Menger定理]]）
- **Harary图最优性** — $H_{k,n}$ 是边数最少的 $n$ 阶 $k$ 连通图（[[Harary图]]）
