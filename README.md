# 图论笔记知识库

> 基于电子科技大学《图论》课程的 Obsidian 关系型笔记仓库。

本仓库以**卡片盒笔记法（Zettelkasten）**为理念，通过 Obsidian 的双向链接（Wikilink）、图谱视图（Graph View）与 Dataview 插件，将图论知识点编织成一张可导航、可查询、可视化的概念网络。

---

## ✨ 核心特性

- **章节体系**：覆盖课程 8 个核心章节，从图的基本概念到有向图。
- **原子化笔记**：每个概念、定理、算法独立成页（`wiki/` 下 70+ 页面），便于复用与链接。
- **双向关联**：通过 `[[Wikilink]]` 建立概念间的引用关系，自动生成知识图谱。
- **MOC 导航**：以 Map of Content 汇总定理（[[MOC-定理]]）与算法（[[MOC-算法]]），快速定位。
- **模板化工作流**：提供统一的笔记模板（概念、定理、算法、问题），保证格式一致。
- **Dataview 动态索引**：自动统计概念分布、最近更新、反向链接等。

## 🚀 使用方式

### 1. 克隆仓库

```bash
git clone <repo-url>
cd Graph_Theory
```

### 2. 用 Obsidian 打开

1. 打开 Obsidian → **打开本地仓库** → 选择本文件夹。
2. 若提示信任插件，请启用以支持 Dataview、Wikilink 等功能。

### 3. 开始探索

- **总导航**：打开 `index.md` 查看课程全貌。
- **图谱漫游**：按 `Ctrl+G`（或 `Cmd+G`）打开 Graph View，直观浏览概念网络。
  - 🔴 红色 = 章级 MOC
  - 🔵 蓝色 = 概念 / 定理页面
  - 🟢 绿色 = 算法页面
  - 🟡 黄色 = 经典问题页面
- **定理与算法速查**：查看 `MOC-定理.md` 与 `MOC-算法.md`。

## 📝 笔记规范

新建笔记时，建议从 `templates/` 中选择对应模板：

- **概念**：定义、符号、例子、与其他概念的关系。
- **定理**：命题、证明思路、应用、依赖的前置定理。
- **算法**：输入输出、步骤、复杂度、正确性说明、例题。
- **问题**：问题描述、分析、解答、相关知识点。

Frontmatter 中建议维护 `tags`、`category`、`chapter` 等字段，以便 Dataview 自动索引。
