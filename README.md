# 图论笔记知识库

> 基于电子科技大学《图论》课程的 Obsidian 关系型笔记仓库。

本仓库以**卡片盒笔记法（Zettelkasten）**为理念，通过 Obsidian 的双向链接（Wikilink）、图谱视图（Graph View）与 Dataview 插件，将图论知识点编织成一张可导航、可查询、可视化的概念网络。

![[file-20260423213159370.png]]

---

## ✨ 核心特性

- **章节体系**：覆盖课程 8 个核心章节（第1–7章、第9章），从图的基本概念到有向图。
- **原子化笔记**：128 个独立概念页（`wiki/`），每个概念、定理、算法独立成页，便于复用与链接。
- **双向关联网络**：每个 wiki 页面通过 `related` 字段建立 3–8 个跨章节关联，形成概念知识图谱。
- **三层导航体系**：
  - **章节 MOC**（8个）：按课程顺序组织
  - **主题 MOC**（8个）：按数学主题跨章汇总（连通性、着色、匹配、树、遍历、定理、算法、特殊图）
  - **特殊图 MOC**：8 类考试重点特殊图汇总（偶图、正则图、完全图、树、平面图、Euler图、Hamilton图、有向图）
- **定理/算法速查**：[[MOC-定理]] 与 [[MOC-算法]] 汇总全部章节内容，附关系图与对比表。
- **模板化工作流**：统一的笔记模板（概念、定理、算法、问题），保证格式一致。
- **Dataview 动态索引**：自动统计概念分布、最近更新、反向链接等。

## 🚀 使用方式

### 1. 克隆仓库

```bash
git clone <repo-url>
cd Graph_Theory
```

### 2. 用 Obsidian 打开

1. 打开 Obsidian → **打开本地仓库** → 选择本文件夹。
2. 若提示信任插件，请启用以支持 Dataview、Wikilink 等功能。

### 3. 开始探索

- **总导航**：打开 `index.md` 查看课程全貌与主题入口。
- **图谱漫游**：按 `Ctrl+G`（或 `Cmd+G`）打开 Graph View，直观浏览概念网络。
  - 🔴 红色 = 章级 MOC / 主题 MOC
  - 🔵 蓝色 = 概念页面
  - 🟣 紫色 = 定理页面
  - 🟢 绿色 = 算法页面
  - 🟡 黄色 = 经典问题页面
- **特殊图速查**：查看 [[MOC-特殊图]]，按图类型快速定位考点。
- **定理与算法速查**：查看 [[MOC-定理]] 与 [[MOC-算法]]。

## 🛠️ 技术栈

- [Obsidian](https://obsidian.md/) — 本地优先的 Markdown 知识库
- **Dataview** — 基于 YAML frontmatter 的动态查询与汇总
- **Breadcrumbs** — 页面顶部面包屑层级导航
- **Graph View** — 原生知识图谱可视化（全局图与子图颜色组一致）
- **Juggl** — 交互式 Cytoscape.js 图谱视图（可选增强）
- **Templater** — 模板化笔记工作流
- **CSS Snippets** — 自适应行宽等阅读体验优化

## 📝 笔记规范

新建笔记时，建议从 `templates/` 中选择对应模板：

- **概念**：定义、符号、例子、与其他概念的关系。
- **定理**：命题、证明思路、应用、依赖的前置定理。
- **算法**：输入输出、步骤、复杂度、正确性说明、例题。
- **问题**：问题描述、分析、解答、相关知识点。

Frontmatter 中维护 `chapter`、`category`、`tags`、`related` 等字段：
- `chapter` — 纯字符串，供 Dataview 筛选
- `parent` — Wikilink 格式，供 Breadcrumbs 建立层级
- `related` — 与本概念关联的其他 wiki 页面（3–8个）
