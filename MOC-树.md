# MOC-树

> 图论中**树结构与优化**主题的跨章节汇总，涵盖无向树（第2章）和有向树（第9章），以及生成树、最优树等优化问题。

## 概览

树是连通无圈图，是图论中研究最透彻的结构类。从第2章的基本树理论到第9章的有向树与编码理论，树贯穿图论多个分支。

## 第2章：无向树

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[树]] | 连通无圈图 | ★★★ 六等价刻画 |
| [[森林]] | 不连通的无圈图 | ★★ |
| [[树的性质]] | $m=n-1$、至少两片树叶等 | ★★★ |
| [[树的中心]] | 离心率最小的顶点集 | ★★ |
| [[树的形心]] | 使分支最大顶点数最小的点 | ★★ |

### 生成树

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[生成树]] | 连通图的生成子图且为树 | ★★★ |
| [[破圈法]] | 逐次破圈求生成树 | ★★ |
| [[矩阵树定理]] | Kirchhoff 定理，用行列式计数 | ★★★ |
| [[Cayley公式]] | $\tau(K_n) = n^{n-2}$ | ★★★ |

### 最小生成树

| 概念 | 说明 | 考试重点 |

## 重要定理速查

| 定理 | 内容 | 位置 |
|------|------|------|
| 树的等价刻画 | 六条等价条件（连通无圈、$m=n-1$、唯一路等） | [[树的性质]] |
| 中心定理 | 树的中心由1点或2相邻点组成 | [[树的中心]] |
| 形心定理 | 树的形心由1点或2相邻点组成 | [[树的形心]] |
| 生成树存在定理 | 每个连通图至少包含一棵生成树 | [[生成树]] |
| 矩阵树定理 | $\tau(G) = \frac{1}{n}\lambda_1\cdots\lambda_{n-1}$ | [[矩阵树定理]] |
| Cayley公式 | $\tau(K_n) = n^{n-2}$ | [[Cayley公式]] |
| 完全 $m$ 元树公式 | 树叶数 = $(m-1)i + 1$ | [[m元树]] |

## 与特殊图的关联

- **偶图**：所有树都是偶图
- **平面图**：所有树都是平面图
- **完全图**：Cayley公式计数 $K_n$ 的生成树
- **正则图**：树的度数分布与正则图形成对比

## 与遍历问题的关联

- 树的遍历（先根、中根、后根）是图遍历的基础
- 生成树是分析图结构的重要工具
- Huffman树是最优前缀编码的基础
