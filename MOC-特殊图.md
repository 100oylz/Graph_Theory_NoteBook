## 偶图（二分图）

> **偶图**（Bipartite Graph）是最重要的一类特殊图，其顶点集可划分为两个不相交子集，使得每条边的两个端点分别属于不同子集。

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[偶图]] | 顶点集二分类 $(X,Y)$，边跨集合 | ★★★ 定义、完全偶图 $K_{m,n}$ |
| [[偶图判定定理]] | 图是偶图 ⟺ 不含奇圈 | ★★★ 充要条件 |
| [[偶图匹配存在性判定定理]] | Hall 定理：匹配覆盖 $X$ ⟺ $|N(S)|\geq|S|$ | ★★★ |
| [[偶图的边色数]] | König 定理：偶图 $\chi' = \Delta$ | ★★★ |
| [[konig定理]] | 偶图中：最大匹配数 = 最小点覆盖数 | ★★★ |
| [[匈牙利算法]] | 求偶图最大匹配的多项式算法 | ★★★ |
| [[Kuhn-Munkres算法]] | 赋权完全偶图的最优匹配 | ★★ |
| [[最优匹配]] | 权和最大的完美匹配 | ★★ |
| [[点覆盖]] | 覆盖所有边的顶点子集 | ★★ |
| [[邻集]] | 邻点集合 $N(S)$ | ★★ |

### 特殊图交集
- 偶图 ∩ 正则图：$K_{n,n}$ 是 $n$-正则偶图
- 偶图 ∩ 完全图：完全偶图 $K_{m,n}$
- 偶图 ∩ 树：所有树都是偶图

## 正则图

> **正则图**（Regular Graph）是所有顶点度数相同的图，在因子分解和着色理论中有核心地位。

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[度序列]] | 顶点度数的非增序列 | ★★★ 可图性判定（Havel-Hakimi） |
| [[握手定理]] | $\sum d(v) = 2m$，奇点个数为偶 | ★★★ |
| [[1-因子分解]] | 1-正则生成子图的分解 | ★★ 完全图 $K_{2n}$ 的1-因子分解 |
| [[2-因子分解]] | 2-正则生成子图的分解 | ★★ 偶数度正则图的2-因子分解 |
| [[3正则图的完美匹配]] | 3正则图无割边则有完美匹配 | ★★★ Petersen定理 |
| [[奇阶正则图的边色数]] | 奇阶 $\Delta$-正则图：$\chi' = \Delta + 1$ | ★★★ 类1/类2判定 |

### 特殊图交集
- 正则图 ∩ 完全图：$K_n$ 是 $(n-1)$-正则图
- 正则图 ∩ 平面图：Platonic图（正多面体图）是正则平面图
- 正则图 ∩ Euler图：偶数度正则连通图是 Euler 图

---

## 完全图

> **完全图**（Complete Graph）$K_n$ 是边数最多的简单图，在计数、极值和着色问题中作为基准对象。

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[完全图]] | 任意两点相邻，$m = \frac{n(n-1)}{2}$ | ★★★ |
| [[Cayley公式]] | $\tau(K_n) = n^{n-2}$ | ★★★ 生成树计数 |
| [[Turán定理]] | 不含 $K_{l+1}$ 的图最大边数 | ★★★ 极图问题 |
| [[极图]] | Turán 图 $T_{l,n}$，完全 $l$ 部图 | ★★★ |
| [[旅行售货员问题]] | $K_n$ 赋权图中的最短 Hamilton 圈 | ★★★ |

### 特殊图交集
- 完全图 ∩ 树：仅 $K_1, K_2$ 是树
- 完全图 ∩ 平面图：仅 $K_1$–$K_4$ 是平面图（$K_5$ 是非平面图基图）
- 完全图 ∩ Euler图：$K_n$ 当 $n$ 为奇数时是 Euler 图
- 完全图 ∩ Hamilton图：$K_n$（$n\geq 3$）都是 Hamilton 图

## 树

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[树]] | 连通无圈图 | ★★★ 六等价刻画 |
| [[森林]] | 不连通的无圈图 | ★★ |
| [[树的性质]] | 边数 $m=n-1$，至少两片树叶等 | ★★★ |
| [[树的中心]] | 由1点或2相邻点组成 | ★★ |
| [[树的形心]] | 由1点或2相邻点组成 | ★★ |
| [[生成树]] | 连通图的生成子图且为树 | ★★★ |
| [[破圈法]] | 求生成树的算法 | ★★ |
| [[矩阵树定理]] | Kirchhoff 定理，用代数方法计数 | ★★★ |
| [[最小生成树]] | 权和最小的生成树 | ★★★ |
| [[Kruskal算法]] | 贪心选边，$O(m\log m)$ | ★★★ |
| [[Prim算法]] | 贪心加点，$O(n^2)$ | ★★★ |

### 有向树（第9章）

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[有向树]] | 以根为起点的有向树 | ★★ |
| [[根树]] | 根、父亲、儿子、树叶、分支点 | ★★★ |
| [[有序树]] | 规定儿子次序的根树 | ★★ |
| [[子树]] | 以某顶点为根的子树 | ★★ |

| [[Huffman算法]] | 构造最优树的贪心算法 | ★★★ |

### 特殊图交集

- 树 ∩ 平面图：所有树都是平面图
- 树 ∩ Euler图：仅 $K_1$（平凡树）
- 树 ∩ Hamilton图：仅 $K_1, K_2$

## 平面图

> **平面图**（Planar Graph）是可画在平面上而使边仅在端点处相交的图，在电路设计、地图着色等领域有重要应用。

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[平面图]] | 可平面嵌入的图 | ★★★ |
| [[Euler公式]] | $n - m + \phi = 2$（连通） | ★★★ |
| [[Kuratowski定理]] | 平面图 ⟺ 不含 $K_5$ 或 $K_{3,3}$ 的同胚 | ★★★ 判定定理 |
| [[瓦格纳定理]] | 平面图 ⟺ 不含 $K_5$ 或 $K_{3,3}$ 的收缩子图 | ★★★ |
| [[对偶图]] | 面对应顶点、边对偶 | ★★★ |
| [[五色定理]] | 平面图 $\chi \leq 5$（Heawood） | ★★★ |
| [[凸多面体与平面图]] | Platonic 图：仅5种正多面体 | ★★ |
| [[Platonic定理]] | 正多面体仅有五种 | ★★ |
| [[平面性算法]] | DMP 算法判定平面性 | ★★ |
| [[哈密尔顿图的平面性判定]] | 利用 Hamilton 圈简化判定 | ★★ |

### 特殊图交集
- 平面图 ∩ 完全图：仅 $K_1$–$K_4$
- 平面图 ∩ 偶图：$K_{3,3}$ 是非平面图基图
- 平面图 ∩ 树：所有树都是平面图
- 平面图 ∩ Euler图：对偶图的关系

## Euler图

> **Euler图** 是可以遍历每条边恰好一次的图，与"一笔画"问题直接相关。

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[Euler图]] | 存在 Euler 回路的图 | ★★★ |
| [[Euler回路]] | 经过每条边恰好一次的闭迹 | ★★★ |
| [[Euler迹]] | 经过每条边恰好一次的迹（非闭） | ★★★ |
| [[Fleury算法]] | 求 Euler 回路/迹的 $O(m)$ 算法 | ★★★ |
| [[中国邮递员问题]] | 每条边至少走一遍的最短闭途径 | ★★★ |

### 判定条件
- **连通图是 Euler 图 ⟺ 每个顶点度数为偶数 ⟺ 边集可划分为边不重的圈**
- **连通图有非封闭 Euler 迹 ⟺ 恰有两个奇度顶点**

### 特殊图交集
- Euler图 ∩ 正则图：偶数度正则连通图
- Euler图 ∩ 完全图：$K_n$（$n$ 为奇数）
- Euler图 ∩ 树：仅 $K_1$

---

## Hamilton图

> **Hamilton图** 是可以遍历每个顶点恰好一次的图，是 NP-complete 问题，考试重点在于充分条件。

### 核心概念

| 概念 | 说明 | 考试重点 |
|------|------|---------|
| [[Hamilton图]] | 存在 Hamilton 圈的图 | ★★★ |
| [[Hamilton路]] | 经过每个顶点恰好一次的路 | ★★★ |
| [[Dirac定理]] | $\delta \geq n/2$ ⟹ H 图 | ★★★ |
| [[Ore定理]] | 不相邻 $d(u)+d(v)\geq n$ ⟹ H 图 | ★★★ |
| [[Chvátal定理]] | 基于度序列的闭包条件 | ★★★ |
| [[度极大非Hamilton图]] | $C_{m,n}$ 图族，极图特征 | ★★ |
| [[旅行售货员问题]] | 最短 Hamilton 圈（TSP） | ★★★ |

### 判定条件
- **必要条件**：$\forall S\neq\varnothing, \omega(G-S)\leq|S|$（但非充分）
- **充分条件**：Dirac、Ore、Chvátal 定理（闭包引理）

### 特殊图交集
- Hamilton图 ∩ 完全图：$K_n$（$n\geq 3$）
- Hamilton图 ∩ 正则图：充分条件（度数条件）
- Hamilton图 ∩ 平面图：平面性判定简化

| [[基础图]] | 忽略方向得到的无向图 | ★★ |
| [[定向图]] | 无向图的定向方式 | ★★ |
| [[出度与入度]] | $d^+(v), d^-(v)$ | ★★★ 握手定理推广 |
| [[有向图的连通性]] | 强连通、单向连通、弱连通 | ★★★ |
| [[强连通分支]] | 极大强连通子图 | ★★★ |
| [[单向连通分支]] | 极大单向连通子图 | ★★ |
| [[弱连通分支]] | 极大弱连通子图 | ★★ |
| [[有向树]] | 以根为起点的有向树 | ★★ |
| [[根树]] | 根、父亲、儿子、祖先、后代 | ★★★ |
| [[m元树]] | 分支点至多 $m$ 个儿子 | ★★★ |
| [[最优树]] | 带权路径长度最小的二元树 | ★★★ |

- 强连通有向图 ∩ Euler图：强连通且各顶点出度=入度

## 学习建议

1. **先掌握定义**：每种特殊图的严格定义是考试的基础
2. **再记判定定理**：偶图（无奇圈）、Euler图（全偶度）、平面图（无 $K_5/K_{3,3}$ 同胚）、Hamilton图（充分条件）
3. **关注交集性质**：如"树是偶图"、"$K_n$ 是 $(n-1)$-正则图"等结论
4. **结合算法理解**：匈牙利算法（偶图）、Fleury算法（Euler图）、Huffman算法（最优树）
