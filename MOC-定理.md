## 定理关系图

### Euler ⟷ Hamilton 对比

```
Euler图                      Hamilton图
  │                            │
  ├─ 遍历每条边                ├─ 遍历每个顶点
  ├─ P问题（多项式时间）        ├─ NP-complete
  ├─ 充要条件（度数）           ├─ 仅有充分条件
  └─ Fleury算法               └─ 无多项式算法
```

### 着色上界链条

```
一般图:    χ ≤ Δ + 1      （平凡上界）
            ↓
Brooks:    χ ≤ Δ          （排除完全图和奇圈）
            ↓
平面图:    χ ≤ 5          （五色定理）
            ↓
四色定理:  χ ≤ 4          （平面图，已证明）
```

### 匹配定理层次

```
一般图:
  Berge定理 ──→ 最大匹配判定
  Tutte定理 ──→ 完美匹配充要条件

偶图（更强结果）:
  Hall定理 ──→ 匹配存在性
  König定理 ──→ 最大匹配 = 最小覆盖
```
