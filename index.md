## 学习路径建议

### 按章节顺序
```
第1章 → 第2章 → 第3章 → 第4章 → 第5章 → 第6章 → 第7章 → 第9章
```

### 按主题深入
- **连通性**：[[MOC-连通性]] ← 第1章 + 第3章
- **着色理论**：[[MOC-着色]] ← 第6章 + 第7章
- **匹配与因子**：[[MOC-匹配]] ← 第5章
- **树与优化**：[[MOC-树]] ← 第2章 + 第9章
- **遍历问题**：[[MOC-遍历]] ← 第4章

### 考试重点（特殊图）
- [[偶图]] → 判定定理、匹配、边色数
- [[正则图]] → 因子分解、完美匹配、边色数
- [[完全图]] → Cayley公式、Turán定理、TSP
- [[树]] → 等价刻画、生成树、Huffman
- [[平面图]] → Euler公式、Kuratowski定理、五色定理
- [[Euler图]] → 判定条件、Fleury算法
- [[Hamilton图]] → Dirac/Ore/Chvátal定理
