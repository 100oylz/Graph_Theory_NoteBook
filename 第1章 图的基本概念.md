# 第1章 图的基本概念

## 本章概览

本章是图论课程的入门基础，系统介绍图的基本概念、运算、连通性、最短路算法、代数表示以及极图理论。掌握本章内容为后续章节的学习奠定坚实基础。

## 核心概念

### 1.1 图和简单图
- [[图]] — 顶点集与边集构成的有序对
- [[图的同构]] — 图之间的结构等价关系
- [[完全图]] — 任意两点均相邻的简单图 $K_n$
- [[偶图]] — 二部图及其完全形式 $K_{m,n}$
- [[补图]] — 相对于完全图的补，包含自补图概念
- [[度序列]] — 顶点度数的序列及其可图性判定
- [[图的频序列]] — 度频数构成的序列

### 1.2 子图与图的运算
- [[子图]] — 真子图、生成子图、导出子图
- [[图的运算]] — 并、交、差、对称差、联图、积图、合成图

### 1.3 路与连通性
- [[路]] — 途径、迹、路、圈、距离与直径
- [[图的连通性]] — 连通图、连通分支及其性质
- [[偶图判定定理]] — 不含奇圈的充要条件

### 1.4 最短路及其算法
- [[最短路]] — 赋权图中的最短路径概念
- [[最短路算法]] — Dijkstra 顶点标号法

### 1.5 图的代数表示及其特征
- [[图的代数表示]] — 邻接矩阵与关联矩阵概述
- [[邻接矩阵]] — $A^k$ 与途径计数定理
- [[关联矩阵]] — 点边关联关系及其应用（全一问题）

### 1.6 极图
- [[极图]] — $l$ 部图、完全 $l$ 部图、Turán 图 $T_{l,n}$
- [[Turán定理]] — 不含 $K_{l+1}$ 的图的最大边数

## 重要定理
- [[握手定理]] — $\sum d(v) = 2m$，奇点个数为偶数
- [[自补图定理]] — 自补图的阶数 $n \equiv 0, 1 \pmod{4}$
- [[偶图判定定理]] — 图是偶图当且仅当不含奇圈
- [[Turán定理]] — 不含 $K_{l+1}$ 的 $n$ 阶图边数不超过 $m(T_{l,n})$
